### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/elvintern/ts-dotnet-petstore.git
   ```
2. Install NPM packages for the Client-side
   ```sh
   npm install
   ```
3. Install NuGet Package for the Server-side
   ```sh
   dotnet restore
   ```
4. Run the server
   ```sh
   dotnet run
   ```
5. Run the client
   ```sh
   npm run start
   ```
